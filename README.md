# No-Display-Cutout
Want to get rid of the cutout on your Android deviceâ€™s screen? This module will allow you to hide it. The module was initially designed for custom ROMs where the cutout was displayed incorrectly and was larger than the camera. However, the module will also help you run games in full-screen mode, ignoring the cutout.
# How to install
This module needs Magisk or Magisk Delta (root access). Go to the Modules tab and pick the ZIP file.
